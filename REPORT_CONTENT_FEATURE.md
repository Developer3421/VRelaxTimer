# Report AI Content Feature

## Overview
The Report AI Content feature allows users to report inappropriate, concerning, or problematic content generated by the AI assistant. This helps maintain content quality and provides a feedback mechanism.

## Access
- **Location**: Main window, bottom section
- **Button**: "Report AI" / "Скарга" / "KI melden" / etc.
- **Position**: Next to "Agreement" button, after "Language" button

## Window Features

### Input Area
- **Multi-line TextBox**: Accepts detailed descriptions
- **Auto-wrapping**: Text wraps automatically
- **Scrollable**: For longer reports
- **No character limit**: Write as much detail as needed

### Buttons

#### SAVE REPORT
- Opens Windows Save File Dialog
- Default filename: `AI_Content_Report_YYYY-MM-DD_HHMMSS.txt`
- Default filter: Text files (*.txt)
- Saves report with metadata header

#### CLOSE
- Simply closes the window
- No data is saved
- Returns to main application

## Report Format

```
AI Content Report
Generated: 2024-12-28 15:30:45
Application: VRelaxTimer
Language: uk

--------------------------------------------------

[User's report text goes here]

--------------------------------------------------
End of Report
```

## Validations

### Empty Report
- **Check**: Report text cannot be empty
- **Message**: "Please enter a description of the issue before saving."
- **Action**: Shows warning dialog, report not saved

### Save Errors
- **Check**: File write permissions, disk space
- **Message**: Shows error with technical details
- **Action**: User can try different location

## Use Cases

### Inappropriate Content
User notices AI generated offensive or inappropriate suggestions:
1. Click "Report AI" button
2. Describe the problematic content
3. Include context (what prompted the response)
4. Click "SAVE REPORT"
5. Choose save location
6. Report saved for review

### Factual Errors
AI provides incorrect information:
1. Click "Report AI" button
2. Note the incorrect information
3. Provide correct information if known
4. Click "SAVE REPORT"
5. Save for future reference

### Quality Issues
AI response is unhelpful or confusing:
1. Click "Report AI" button
2. Describe why response was problematic
3. Suggest improvements if possible
4. Click "SAVE REPORT"
5. Document the issue

## Privacy & Data

### Local Storage Only
- Reports saved to user's chosen location
- No automatic upload or transmission
- User controls where files are stored
- Complete privacy maintained

### What's Included
- Timestamp of report creation
- Application name (VRelaxTimer)
- User's selected language
- User's description of the issue

### What's NOT Included
- Actual AI conversation history
- User's personal information
- System information
- Network data

## Best Practices

### Writing Effective Reports
1. **Be Specific**: Include exact phrases or content
2. **Provide Context**: What was the conversation about?
3. **Describe Impact**: Why is this concerning?
4. **Suggest Solutions**: What would be better?

### Example Report
```
I asked the AI for relaxation techniques, and it suggested 
something that seemed medically unsafe. Specifically, it 
recommended holding breath for extended periods without 
proper guidance. This could be dangerous for users with 
respiratory conditions.

Suggestion: AI should include safety disclaimers when 
suggesting breathing exercises and recommend consulting 
healthcare professionals for medical advice.
```
